# yadegari
Simple full-stack Node.js app for writing memos

## Note
This project was intended to be an example for my Node.JS learning journey and will not hVe further updates.

## Run
- You should have MongoDB installed and running on your host.
- Install Docker
- Run with Docker: (Use rout own port and web address)
```bash

sudo docker run -d --network=host -e PORT=3000 -e WEB_ADDRESS="http://localhost:3000" alirezabrtn/yadegari:latest

```
